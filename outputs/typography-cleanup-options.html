Minimal Cleanup

Inspired by Sakara Life — keep italic as one signature move, tighten everything else

What Changes

~
Italic: keep only in Hero H1 — remove from all other 11 headings. One italic moment per page = signature, not noise.
~
Reduce to 2 weights: font-normal (400) for headings + body, font-medium (500) for eyebrows + buttons. Kill font-semibold and font-bold.
✓
Keep uppercase on eyebrows + buttons + nav as-is. Already matches Sakara's pattern.
✕
Remove text-shadow from hero text elements (already done).

Impact

✓
Smallest change — fixes the worst offender (italic everywhere) without touching layout or button styles. ~30 files affected.

Full Cleanup

Inspired by PANPURI + Cure Hydration — zero italic, zero bold, pure size-and-color hierarchy

What Changes

✕
Remove ALL italic — from every heading, every component. Zero exceptions. Playfair Display Regular only.
~
Exactly 2 weights: font-normal (400) everywhere. font-medium (500) only for buttons + nav links. Nothing else.
~
Uppercase only on: eyebrow labels + buttons. Reduce Footer labels from uppercase to sentence case.
~
Heading hierarchy by size only: H1 display, H2 large, H3 medium — no weight or style variation between them.

Impact

~
Significant cleanup. Removes all decorative typography. Site becomes very calm and confident. ~35 files affected. Most aligned with PANPURI and Cure.

C

Luxury Quiet

Inspired by Grown Alchemist — zero italic, zero uppercase on buttons, hierarchy through scale alone

What Changes

✕
Remove ALL italic — same as Option B. Playfair Regular only.
~
2 weights: font-normal (400) for everything. font-medium (500) only for eyebrow labels.
~
Buttons: sentence case. No uppercase, no letter-spacing. Just clean text in a pill. Like Grown Alchemist's "Shop" buttons.
✓
Uppercase only on eyebrows. Everything else is sentence case. The quietest possible typography.
~
Nav links: sentence case. No uppercase, no tracking. Just weight 400 + color for active state.

Impact

~
Most dramatic shift. Site feels like a high-end apothecary. Very quiet, very confident. ~40 files affected. Closest to Grown Alchemist's approach.
